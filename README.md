## How to write a rails acts_as_xxx plugin

##### [Create a plugin with rails cmd](http://guides.rubyonrails.org/plugins.html)

```ruby
rails plugin new acts_as_votable
cd acts_as_votable

# edit gemspec
```

##### Test it

```ruby
rake -T
rake test
```
